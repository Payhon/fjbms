## 📝 项目信息

### 技术栈概览

**后端**:
- **语言**: Golang 1.23+
- **框架**: Gin
- **数据库**: PostgreSQL + TimescaleDB
- **缓存**: Redis
- **消息**: MQTT (GMQTT/VerneMQ)

**前端**:
- **框架**: Vue.js 3 + Vite
- **语言**: TypeScript
- **UI库**: Element Plus

**移动端**:
- **框架**: Uni-app
- **平台**: iOS, Android, 微信小程序, H5

### 核心功能

- ✅ 设备管理（多协议接入）
- ✅ 数据采集与展示
- ✅ 告警管理
- ✅ 场景自动化
- ✅ 可视化大屏
- ✅ OTA固件升级
- ✅ 多租户支持
- ✅ 权限管理
