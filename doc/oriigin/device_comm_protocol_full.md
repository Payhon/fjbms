# 云平台参数修改与OTA协议
## 一、基础指令操作
### 1.1 同步BMS时间
同步BMS时间需发送**4个字节时间戳**，对应寄存器地址为`0X57C~0X57D`，备注如下：
- H表示高字节，L表示低字节；
- 保留或不支持某地址功能，用FF填充。

### 1.2 获取BMS板UUID
| 配置项 | 取值/说明 |
| ---- | ---- |
| 功能码 | 0XFF |
| 起始地址 | 0x0000 |
| 寄存器数量 | 0x0008 |
| 返回数据 | 16字节UUID |

### 1.3 查询UUID指令示例
- **发送指令**：7F 55 FE 00 FF 00 00 00 08 15 C9 FD
- **返回数据**：7F 55 01 FE FF 10 41 50 35 31 38 37 39 01 00 B5 7E 4C 47 02 50 78 09 C7 FD
- 注：示例中`41 50 35 31 38 37 39 01 00 B5 7E 4C 47 02 50 78`为16位UUID码。

### 1.4 仪表地址（0XFC）操作
**应用场景**：小程序已连接某电池/仪表后，再次连接新电池MAC时，发送6个新MAC地址用于仪表变更连接。
- **发送指令**：7F 55 FE FC 10 00 00 00 03 06 BC EC 3A 4A AB AC CRCL RCH FD
- **返回指令**：7F55FC FE1000000003 CRCL CRCHED

## 二、寄存器配置
### 2.1 系统寄存器（上位机最高权限配置）
| 地址 | 名称 | 字节描述 | 字节长度 | 读/写 | 单位 | 偏移 | 分辨率 | 描述 |
| ---- | ---- | ---- | ---- | ---- | ---- | ---- | ---- | ---- |
| - | 高低串数配置 | H-高串数(一般为0)；L-低串数 | 2 | RW | - | - | - | 高串+低串等于串数总和，支持改串数功能 |

## 三、工厂指令操作
### 3.1 功能键操作寄存器（0X57A~0X57B）
4字节寄存器，BIT位定义如下：
- BIT28：功能4.1，1-打开，0-无操作，2-功能1关闭；
- BIT19：清除保护状态；
- BIT18：擦除循环次数，1-擦除，0-无操作；
- BIT1：保留；
- BIT0：一键掉电，1-掉电，0-无操作。

## 四、通信基础规范
1. 数据存储/传输：**大端模式**；
2. 物理层：UART、485，默认波特率**9600**，格式**8N1**；
3. 地址定义：上位机地址**0XFE**，目标地址单机默认**0x01**；
4. 校验算法：**MODBUS CRC**，校验数据包含：来源地址、目标地址、指令ID、DATA数量、所有DATA。
