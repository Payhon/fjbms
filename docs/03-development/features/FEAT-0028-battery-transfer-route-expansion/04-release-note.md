## 1. 发布内容
- 新增经销商回 PACK、经销商到门店、门店回经销商三条调拨路径。
- 管理端调拨弹窗只展示当前账号可选的目标机构类型。
- 组织选项接口支持按上级链路返回 PACK / 经销商选项。

## 2. 影响范围
- `backend/` 电池生命周期调拨服务、组织选项接口
- `frontend/` BMS 管理 > 电池列表

## 3. 升级步骤
1. 部署后端代码。
2. 部署前端代码。
3. 用经销商和门店账号分别回归新增调拨链路。

## 4. 回滚步骤
1. 回滚 FEAT-0028 对应后端与前端提交。
2. 保留已产生的转移记录，不做历史数据回退。

## 5. 已知问题
