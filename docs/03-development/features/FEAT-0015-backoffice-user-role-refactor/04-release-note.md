# FEAT-0015 后台用户体系与角色权限重构 - 发布说明

- status: in_progress
- owner: payhon
- last_updated: 2026-03-13
- related_feature: FEAT-0015
- version: v0.1.0

## 1. 发布内容
- 新增后台用户主账号/子账号定义：`users.is_main`
- 新增角色业务表：`user_roles`、`role_permissions`
- 统一租户识别口径为租户主账号
- 后台账号管理支持租户后台账号、机构主账号、机构员工账号
- 后台角色管理支持作用域字段与按范围加载权限树
- 菜单迁移到 `系统管理 > 后台账号管理 / 后台角色管理`

## 2. 影响范围
- 后台账号管理
- 后台角色管理
- 登录菜单/按钮权限
- 租户识别与默认租户逻辑

## 3. 升级步骤
1. 执行 `backend/sql/45.sql`
2. 发布后端服务
3. 发布前端管理端
4. 抽样检查租户主账号、机构主账号、角色权限和菜单显隐是否符合预期

## 4. 回滚步骤
1. 回滚前端管理端版本
2. 回滚后端服务版本
3. 如需回滚数据库结构，先导出 `users.is_main`、`user_roles`、`role_permissions` 相关数据后执行结构级回退
