# FEAT-0026 UniApp 账号注销 - 发布说明

- status: in_progress
- owner: payhon
- last_updated: 2026-03-26

- version: v0.1.0

## 1. 发布内容
- UniApp 设置页新增“账号注销”功能。
- 注销前新增不可恢复提示与当前密码验证。
- 后端新增终端用户自助注销接口与数据清理逻辑。

## 2. 影响范围
- `backend/` APP 鉴权接口与用户清理逻辑
- `fjbms-uniapp/` 我的 > 设置 页面

## 3. 升级步骤
1. 部署后端最新代码。
2. 发布 UniApp 新版本。
3. 使用终端用户账号验证“确认提示 -> 输入密码 -> 注销成功退出登录”链路。

## 4. 回滚步骤
1. 回滚 FEAT-0026 相关后端与 UniApp 代码。
2. 客户端隐藏“账号注销”入口，恢复仅保留退出登录。

## 5. 已知问题
- 本次仅支持终端用户自助注销，不支持业务账号/租户账号。
