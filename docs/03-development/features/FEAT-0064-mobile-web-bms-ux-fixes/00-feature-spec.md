# FEAT-0064 移动端与 Web BMS 展示修正 - 功能规格

- status: review
- owner: payhon
- last_updated: 2026-07-07
- related_feature: FEAT-0064
- version: v0.1.0

## 1. 背景与目标
- PACK 厂微信小程序进入首页或登录页时，运行配置加载前后仍可能看到默认富嘉 BMS Banner/Logo，造成品牌闪现。
- 移动端 BMS 保护状态列表中“开启/关闭”和英文 `On/Off` 需要统一成 `ON/OFF`。
- Web 管理端电池详情 BMS 面板的电芯电压值当前竖排显示，不利于快速读取。
- 目标是在不修改后端接口、数据库、权限、协议或设备数据解析的前提下修正上述展示问题。

## 2. 范围
### In Scope
- UniApp 微信小程序首页 Banner 和登录页 Logo 的默认品牌图兜底策略。
- UniApp BMS 保护状态 ON/OFF 文案。
- Web BMS 面板电芯 TAB 电压值横向展示。
- FEAT-0064 文档与看板同步。

### Out of Scope
- 不新增或修改后端接口、数据库迁移、权限项或设备通信协议。
- 不调整 PACK 小程序配置表单、图片上传和运行配置接口。
- 不改变 Web 电芯电压数据来源、精度、单位和高低电压/均衡状态逻辑。

## 3. 用户价值
- PACK 小程序冷启动和进入页面时不暴露富嘉默认品牌资产。
- BMS 保护状态在中英文环境下统一显示 `ON/OFF`，减少歧义。
- Web 管理端查看电芯电压时更清晰，适合多串电芯横向扫读。

## 4. 验收标准
1. PACK 小程序运行配置加载中、失败、字段缺失或未配置图片时，首页和登录页不显示默认富嘉 Banner/Logo。
2. PACK 小程序配置专属 Banner/Logo 后，首页和登录页显示专属图片。
3. 普通租户小程序在运行配置明确返回 `source_type=TENANT` 且 `login_only=false` 时，未配置图片可继续回退默认富嘉图。
4. 移动端 BMS 保护状态展开后，中英文均显示 `ON/OFF`。
5. Web BMS 面板电芯 TAB 下方电压值横向单行显示，多串电芯横向滚动和高低电压/均衡样式保持正常。

## 5. 风险与约束
- 微信小程序 runtime 类型不明时按空白无品牌处理，优先避免默认品牌误闪。
- 小程序首屏行为依赖 AppID 和运行配置接口时序，需要微信开发者工具或真机回归。
- Web 本次只改 CSS 展示方向，需浏览器确认长电芯列表滚动和数值不重叠。

## 6. 回滚方案
- 回滚 `fjbms-uniapp` 中 runtime 默认品牌图兜底 helper、首页/登录页调用和 ON/OFF 文案改动。
- 回滚 Web BMS 面板 `.cell-battery-voltage` 样式改动。
- 同步将 FEAT-0064 看板状态退回并在实施日志记录回滚原因。
