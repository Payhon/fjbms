# FEAT-0061 设备详情充放电控制工厂命令 - 发布说明

- status: review
- owner: payhon
- last_updated: 2026-06-25
- related_feature: FEAT-0061
- version: v0.1.0

## 1. 发布内容
- Web 设备详情高级设置新增 `禁止充电`、`禁止放电`、`允许充放电` 三个工厂命令。
- UniApp 设备详情高级参数出厂配置新增同名三个操作。
- 后台设备参数权限树新增对应 `factory:*` 权限项，可按机构类型授权。

## 2. 影响范围
- 影响 Web BMS 设备详情页。
- 影响 UniApp 设备详情参数设置页。
- 影响后端设备参数权限树接口返回。
- 不影响数据库结构、HTTP 路由或设备协议封装格式。

## 3. 验证建议
- 发布前执行后端权限树单测、Web typecheck 和 UniApp TypeScript 检查。
- 使用受限账号验证三项权限的显隐。
- 在确认安全的测试设备上分别验证三项命令下发。

## 4. 回滚说明
- 如出现按钮误显或命令映射异常，回滚 FEAT-0061 对 Web、UniApp、后端权限树和文档看板的改动。
- 无数据库回滚步骤。
