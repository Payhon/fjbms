## 日志
1. 完成 FEAT-0012 文档建档（规格/设计/实施/测试/发布）。
2. 新增 SQL 补丁 `backend/sql/42.sql`：
   - `attribute_history_datas`、`bms_history_export_jobs` 表与索引。
   - `bms_battery_history` 菜单与 `bms_battery_history_export` 按钮权限。
3. 后端新增 BMS 历史查询与导出接口、异步任务、下载路由与 WS 通知。
4. 后端属性入库链路增加历史明细写入。
5. 前端新增 BMS 历史数据页面（左筛选右表格、长宽表切换、异步导出）。
6. 前端全局头部新增导出任务消息图标、闪烁提醒、Popover 任务表与下载联动。
7. 路由与多语言补齐（zh/en）。
8. 修复 SQL 补丁 `42.sql` 的 `DO $$ ... END $$;` 分隔符，避免执行失败。
9. 修复前端页面元素权限状态读取兼容逻辑，确保按钮权限按 `ui_codes` 正常生效。
10. 优化导出 WS 推送逻辑：Redis 可用时仅走 PubSub，避免同实例重复推送导致前端重复提醒。
11. 优化前端导出消息中心初始化/销毁状态，支持登录态切换后自动重连并刷新未下载任务。
12. 根据测试反馈修复左侧设备筛选数据源：设备列表查询改为 `LEFT JOIN device_batteries`，默认可对齐电池列表展示全部设备。
13. 根据测试反馈优化历史页左侧设备筛选展示：改为小数字分页导航（替换“加载更多”），并调整卡片布局确保分页查询结果稳定可见。
14. 根据测试反馈补充前端接口响应兼容解析（`payload / payload.data / payload.data.data`），修复接口有数据但左侧列表判空的问题。
15. 根据测试反馈优化历史页右侧布局与宽表列名可读性：修复分页区溢出/滚动条，增加宽表字段中文映射与自动翻译规则。
16. 根据测试反馈继续修正右侧表格分页遮挡问题：按内容区总高度重新计算表格可视高度，并将分页区改为固定占位，不再覆盖最后一行数据。
17. 根据测试反馈继续优化左侧设备筛选布局：列表区改为独立滚动容器，分页切换为简易数字模式并固定在底部，不再覆盖设备卡片。
18. 根据测试反馈继续收紧左右区块中间的折叠控制条：中缝宽度缩至 8px，折叠按钮与图标同步缩小，减少对内容区的视觉侵占。
19. 根据测试反馈将中缝折叠按钮进一步弱化为“悬浮细把手”样式：细长胶囊形态、轻边框与浅阴影，仅在 hover 时增强反馈，降低按钮感。
