# FEAT-0021 UniApp 电池仪表盘组件总电压与曲线进度条优化 - 功能规格

- status: review
- owner: payhon
- last_updated: 2026-03-22
- related_feature: FEAT-0021
- version: v0.1.0

## 1. 背景与目标
- 背景：
  - 当前电池详情页顶部仪表盘组件只展示 SOC/SOH，缺少总电压信息。
  - 左右两侧 SOC/SOH 进度条为折线六边形走势，视觉上与外层容器的圆角壳体不一致。
- 目标：
  1. 在仪表盘上方中央缺口区域新增总电压显示。
  2. 将左右进度条改为与外层容器弧度一致的椭圆曲线形态。
  3. 保持现有进度条宽度、渐变方向和整体布局不发生功能性退化。

## 2. 范围
### In Scope
- 调整 `fjbms-uniapp/components/dashboard-gauge/dashboard-gauge.vue` 的绘制逻辑与覆盖层布局。
- 调整 `fjbms-uniapp/pages/device-battery/components/dashboard-tab.vue` 向仪表盘组件传入总电压值。
- 更新中英文文案。

### Out of Scope
- 不调整仪表盘下方其它卡片布局。
- 不改动后端协议与状态解析结构。

## 3. 验收标准
1. 仪表盘顶部中间新增“总电压”与电压值显示，例如 `51.2V`。
2. 总电压显示位于左右进度条顶部缺口的中间区域。
3. 左右两侧进度条从原折线路径改为圆滑椭圆曲线。
4. 新曲线与外层容器弧度方向一致，视觉上贴合外轮廓。
5. 进度条宽度和蓝色渐变风格保持与现状一致。

## 4. 风险与约束
- 该组件同时兼容 canvas 文本绘制和非微信平台的 DOM overlay，需要两套显示同步调整。
- 曲线路径采用采样后的贝塞尔路径绘制，若后续容器外形继续大改，需要同步调整控制点。

## 5. 回滚方案
- 回滚 `dashboard-gauge.vue` 中的总电压 overlay/canvas 文本。
- 回滚左右轨道曲线路径，恢复原折线轨迹点定义。
