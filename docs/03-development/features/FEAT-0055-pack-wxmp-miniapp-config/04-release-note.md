# FEAT-0055 PACK 厂微信小程序配置接入 - 发布说明

- status: review
- owner: payhon
- last_updated: 2026-05-26
- related_feature: FEAT-0055
- version: v0.1.0

## 1. 发布内容
- 支持从 PACK 厂层级配置微信小程序接入信息。
- 支持小程序首页 Banner、关于我们、隐私协议、用户政策按小程序配置。
- 微信小程序端仅保留微信登录注册方式。

## 2. 部署步骤
1. 执行后端 SQL 迁移。
2. 部署后端服务。
3. 部署 WEB 后台。
4. 重新构建并发布微信小程序。
5. 在后台 PACK 厂列表中配置微信 AppID、AppSecret、首页 Banner、关于我们、隐私协议和用户政策。

## 3. 验证重点
- PACK 厂配置保存与回显。
- 小程序微信登录成功。
- Banner 和协议内容按小程序展示。
- 旧 APP/H5 登录入口不受影响。
- 停用配置后，对应小程序登录和运行配置接口不可继续使用。

## 4. 回滚说明
- 回滚后端、WEB 和 UniApp 代码。
- 停止使用 PACK 小程序配置表，保留数据以便后续恢复。
