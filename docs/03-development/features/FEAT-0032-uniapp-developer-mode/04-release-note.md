# FEAT-0032 UniApp 开发者模式 - 发布说明

- status: in_progress
- owner: payhon
- last_updated: 2026-04-03
- related_feature: FEAT-0032
- version: v0.1.0

## 1. 发布内容
- 新增 UniApp 开发者模式全局开关。
- 登录页支持通过顶部 LOGO 连续点击 9 次开启开发者模式。
- 开发者模式下，登录卡片底部显示基座版本号、应用版本号与 API BaseURL。

## 2. 影响范围
- `fjbms-uniapp/` 登录页、全局 store、运行时调试信息工具与多语言字典。

## 3. 升级步骤
1. 部署包含 FEAT-0032 的 UniApp 新版本。
2. 在登录页连续点击顶部 LOGO 9 次，确认开发者模式可开启。
3. 验证登录卡片底部展示的版本号与 API BaseURL 符合当前安装包与环境。

## 4. 回滚步骤
1. 回退 FEAT-0032 对应提交。
2. 清理本地 `__developer_mode_enabled__` 存储键。
3. 重新安装或重启应用，确认登录页恢复为普通展示。

## 5. 已知问题
- 当前版本未提供关闭开发者模式的 UI 入口。
- 不同平台可展示的版本字段可能存在差异，缺失时将显示兜底值。
- 当前尚未补充真机 / 小程序运行态验收记录。
