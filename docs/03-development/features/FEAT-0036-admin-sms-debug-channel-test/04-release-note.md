# FEAT-0036 后台短信验证码调试能力 - 发布说明

- status: review
- owner: payhon
- last_updated: 2026-04-07
- related_feature: FEAT-0036
- version: v0.1.0

## 1. 发布内容
- 后台 `系统管理 > 通知设置 > 短信` 新增“调试”功能。
- `SYS_ADMIN` 调试弹窗新增租户选择，避免误用当前登录租户导致模板判断失真。
- 调试功能支持选择手机号与验证码场景，直连注册验证码完整发送链路。
- 调试结果会显示模板读取、验证码缓存、短信服务发送等步骤，以及模板 ID、签名、Endpoint、RequestId 等诊断信息。

## 2. 影响范围
- 影响模块：`frontend/` 短信配置页，`backend/` 通知配置调试接口与 APP 短信调试服务。
- 不涉及数据库结构变更。
- 不影响移动端正式注册接口协议。

## 3. 升级步骤
- 部署前后端最新代码即可，无需额外迁移。

## 4. 回滚步骤
- 回滚短信调试相关前后端改动与 FEAT-0036 文档记录。

## 5. 已知问题
- 若管理员当前租户与目标 APP 注册租户不一致，调试结果需结合实际租户上下文判断。
