# FEAT-0045 调拨目标收敛与门店上级组织防泄露 - 发布说明

- status: in_progress
- owner: payhon
- last_updated: 2026-04-21
- related_feature: FEAT-0045
- version: v0.1.0

## 1. 发布内容
- 电池调拨规则收紧：
  - 经销商仅可调拨到下级门店；
  - 门店账号不再允许调拨。
- 电池列表调拨弹窗按账号角色收敛目标类型；门店入口保留但禁用。
- 门店管理新增弹窗在经销商场景下，上级组织改为只读当前机构，不再读取完整组织树。
- 后端新增创建门店父子关系校验，阻断绕过前端的越权创建。

## 2. 影响范围
- `backend/`：电池生命周期调拨服务、组织创建服务。
- `frontend/`：电池列表调拨弹窗、门店管理组织弹窗。
- `docs/`：FEAT-0045 新文档，FEAT-0028 口径更新。

## 3. 发布后验证
1. 用经销商账号验证调拨仅门店。
2. 用门店账号验证调拨不可执行且后端拒绝。
3. 用经销商账号新增门店，确认上级组织只读且无组织树泄露。

## 4. 回滚说明
- 回滚 FEAT-0045 对应前后端提交。
- FEAT-0045 不涉及 schema 变更，回滚无需数据迁移。
