# FEAT-0013 移动端设备添加分流、首页视图切换与筛选增强 - 发布说明

- status: in_progress
- owner: payhon
- last_updated: 2026-03-12
- related_feature: FEAT-0013
- version: v0.1.0

## 1. 发布内容
- 移动端设备添加按用户类型分流：
  - 终端用户添加时绑定并激活；
  - 机构用户添加时仅加入“我的设备”。
- “我的”页面新增机构用户首页视图切换图标。
- 首页“我的设备”支持按模式查看：
  - 我添加的设备
  - 终端用户绑定设备
- 首页筛选图标支持真实筛选交互，覆盖终端用户与机构用户场景。

## 2. 影响范围
- `backend/`：设备开通、设备列表、激活日志相关服务与接口。
- `fjbms-uniapp/`：我的页、首页、设备添加流程、筛选弹层与卡片操作。
- `docs/`：FEAT-0013 相关规格、设计、测试与发布文档。
