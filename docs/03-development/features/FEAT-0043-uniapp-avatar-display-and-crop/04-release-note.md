# FEAT-0043 UniApp 头像展示与裁切优化 - 发布说明

- status: in_progress
- owner: payhon
- last_updated: 2026-04-17
- related_feature: FEAT-0043
- version: v0.1.0

## 1. 发布内容
- UniApp “我的”页头像展示改为按容器统一裁剪与居中显示，优化超大头像图的展示稳定性。
- UniApp 设置页“修改头像”右侧补齐当前头像展示，并与“我的”页共用同一头像解析逻辑。
- UniApp 设置页头像修改流程新增裁切步骤，用户可在上传前按 1:1 头像区域调整图片。

## 2. 影响范围
- `fjbms-uniapp/` “我的”页
- `fjbms-uniapp/` “我的 > 设置”页

## 3. 注意事项
- 本次不调整后端接口；头像仍通过现有上传接口和资料更新接口保存。
- 微信小程序授权头像/昵称更新链路保持原状，不纳入本次裁切范围。
