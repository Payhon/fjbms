# FEAT-0033 UniApp 扫描/扫码登录前置校验 - 发布说明

- status: in_progress
- owner: payhon
- last_updated: 2026-04-03
- related_feature: FEAT-0033
- version: v0.1.0

## 1. 发布内容
- 为移动端蓝牙扫描、摄像头扫码及扫码绑定相关页面新增登录前置校验。
- 未登录用户使用上述功能时，将先跳转登录页。

## 2. 影响范围
- `fjbms-uniapp/` 扫描/扫码入口、设备绑定相关页面与仪表临时会话扫码入口。

## 3. 升级步骤
1. 部署包含 FEAT-0033 的 UniApp 新版本。
2. 验证未登录状态下扫描/扫码功能会跳转登录页。
3. 验证已登录状态下扫描/扫码流程正常。

## 4. 回滚步骤
1. 回退 FEAT-0033 对应提交。
2. 重新发版 UniApp。

## 5. 已知问题
- 当前登录成功后不会自动回跳到用户原先尝试进入的扫描/扫码页面。
- 当前尚未补充真机 / 小程序运行态验收记录。
